本项目用于实现Awesome码农登录前端界面，实现后，需要将打包后的生产文件(build/)拷入微服务auth项目下（resources/public）

## 本地测试运行

```bash
$ npm install
$ npm start
```
## 打包
```bash
$ npm run build
```
